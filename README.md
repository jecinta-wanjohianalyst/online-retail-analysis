# Online Retail Analysis

## Overview
This project analyzes an Online Retail dataset using Python.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Objectives
- Clean the data
- Explore customer purchasing behavior
- Analyze sales trends
- Generate business insights

## Key Findings
- The UK generated the highest sales.
- Some products contributed significantly to revenue.
- Customer purchasing patterns varied across countries.
